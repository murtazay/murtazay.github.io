CSC 47200 - Computer Graphics

This was a very fun class. The projects were cool and could be expanded on.
The projects were coded in C++ using the QT Framework and utilized OpenGL to display graphics.
The OpenGL code was written two ways: using C++/GL codes and Shaders.
They have been ported to WebGL.

P Render

This was the first assignment and was really supposed just give a glimpse of the various things you can do with OpenGL.
From the top left to the bottom right the following GL drawmodes are shown:POINTS, LINES, LINE_STRIP, LINE_LOOP, TRIANGLES, TRIANGLE_STRIP, TRIANGLE_FAN, QUADS, POLYGON.
This also illustrates the differences between WebGL and OpenGL. In WebGL you can only draw POINTS, LINES, LINE_STRIP, LINE_LOOP, TRIANGLES, TRIANGLE_STRIP, TRIANGLE_FAN.
