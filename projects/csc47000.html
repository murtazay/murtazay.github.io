CSC 47000 - Image Processing

I took this course as an independent study, so it was very hands-on and detailed. Also as a result I got to do more, learn more and ask more questions as it was pretty much according to my pace. The programs were written twice once using C++ with an image library that was provided by the professor to draw and traverse the image, the other was using C++ for GUI and OpenGL to draw the image. There are quite a few differences between those versions and this WebGL version including a major difference that multiple filters can be applied as opposed to the classroom version which was only one filter at any given time. The output of one filter is the input to the next one. A technical difference is that only contanst loops are allowed in WebGL. This affected blur and sharpen shaders as those shaders had to be made on the fly.
